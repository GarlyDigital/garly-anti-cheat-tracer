# GarlyBot

Riot **Vanguard** ve **Easy Anti-Cheat (EAC)** guncellemelerini izleyen Discord botu.

> **Durum:** Gelistirme asamasinda. Vanguard/EAC izleme ozellikleri yakinda eklenecek.

## Gereksinimler

- Node.js 18+
- Discord bot token ([Developer Portal](https://discord.com/developers/applications))

## Kurulum

```bash
git clone https://github.com/KULLANICI_ADIN/garlybot.git
cd garlybot
npm install
cp ayarlar.example.json ayarlar.json
```

`ayarlar.json` dosyasina Discord bot token'inizi yazin.

## Calistirma

```bash
npm start
```
